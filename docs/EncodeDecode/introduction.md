编码是信息从一种形式或格式转换为另一种形式的过程，用预先规定的方法将文字、数字或其它对象编成数码，或将信息、数据转换成规定的电脉冲信号。编码在电子计算机、电视、遥控和通讯等方面广泛使用。
  
解码，是编码的逆过程。

为什么要编码：
```
 人类语言          -->  "Hello World"
 计算机存储        -->  "???????????"  --> 01010100....1110101010
```

编码：就是人类文字到0、1二进制的映射，反之则叫解码。
 
所以:
 ```
  "Hello world"    -->    01001000...0100
                   编码
  "Hello world"    <--    01001000...0100
                   解码
 ```
狭义上去理解：
 
编码：编成计算机能够识别的、解码：解码是为了人类能读懂
